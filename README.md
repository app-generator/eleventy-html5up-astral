## How to use it

- Build from sources:

```
$ git clone https://github.com/app-generator/eleventy-html5up-astral.git
$ cd eleventy-html5up-astral
$ yarn # install modules 
$ yarn dev # start in development mode

$ # app is running on http://localhost:4000
```
